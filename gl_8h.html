eogllGenVertexArray

This function creates a buffer object.

passing vao to eogllGenBuffer will ensure that the correct vao is bound, and it will unbind it when it's done, this help track down bugs later on if we do a function that requires a vao to be bound, but it's not bound right before, there might be an unnoticed bug later, but if we always unbind it when we're done, we can be sure that that bug will appear right away instead of later on in a distant universe.
