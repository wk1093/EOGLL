See also
eogllSetCursorPos
EogllWindow

This function gets the current cursor pos. This function is different from eogllGetMousePos: eogllGetMousePos is updated by a callback function, so it's updated whenever the mouse moves, but eogllGetCursorPos gets the current cursor pos, so it's only updated when you call this function. eogllGetCursorPos is more CPU intensive than eogllGetMousePos, so you should only use it when you need to.

Sets whether or not the window should close.

Parameters
windowThe window to set
shouldCloseWhether or not the window should close
See also
eogllWindowShouldClose

This function is mostly used when you want to exit the program, but you don't want to have to repeat all of the cleanup code:

while (!eogllWindowShouldClose(window)) {
// do stuff
if (shouldExit) {
eogllSetWindowShouldClose(window, true);
}
// rest of loop
}
// cleanup code HERE in one place
// delete stuff (buffers, shaders, window, etc.)

EOGLL_DECL_FUNC_ND bool eogllWindowShouldClose ( EogllWindow *  window)

Checks if the window should close.

Parameters
windowThe window to check
Returns
Whether or not the window should close
See also
eogllSetWindowShouldClose

This function checks if the window should close (normally this is when the user clicks the close button). Most programs will have a loop that looks something like this:

while (!eogllWindowShouldClose(window)) {
// do stuff
eogllPollEvents(window);
eogllSwapBuffers(window);
}
