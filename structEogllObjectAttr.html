This struct is used to handle object attributes. Object attributes are similar to vertex attributes, but they also store the type of the attribute (where it comes from in the obj file). This allows us to load object files with customizable attributes.

Sample usage:

EogllObjectAttrs objMode = eogllCreateObjectAttrs();
eogllAddObjectAttr(&objMode, GL_FLOAT, 3, EOGLL_ATTR_POSITION);
eogllAddObjectAttr(&objMode, GL_FLOAT, 2, EOGLL_ATTR_TEXTURE);
EogllBufferObject bufferObject = eogllBufferObjectLoad("resources/models/cube.obj", objMode);
